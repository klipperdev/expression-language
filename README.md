Klipper Expression Language Component
========================

The Expression Language component is a set of Expression Language functions.

Features include:

- Current User Id function
- Date Add function
- Date Diff function
- Date Sub function
- Date Time function
- Month Begin function
- Month End function
- Now function
- Today function
- Week Begin function
- Week End function
- Year Begin function
- Year End function
